# Lumo 3D Mobile Texture & Billboard Pack

Enthält über 100 kostenlose, neu generierte Assets für das Lumo-3D-Erlebnis.

Asset-Typen:
- tilebare Material-Texturen + Normal Maps
- Portal-Ringe mit Transparenz
- Emission Maps
- Partikel-Sprites
- Billboard-Objekte
- Sky-Gradienten
- UI-Panels
- Shader-Masken

Diese Assets sind keine echten 3D-Modelle (.glb), sondern 3D-taugliche Texturen/Sprites/Billboards für Godot.
